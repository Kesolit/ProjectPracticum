# Веб-конструктор портфолио для ИТ-специалистов

## Описание проекта
Данный сервис позволяет ИТ-специалистам быстро собирать и публиковать профессиональные портфолио без необходимости программирования по принципу no-code/low-code конструктора. Продукт решает проблему трудоустройства начинающих специалистов, предоставляя инструмент для наглядной демонстрации стека и проектов.

## Цель MVP
Реализация полного цикла:  создание черновика → настройка блоков → выбор темы → публикация → доступ по публичной ссылке.

## Технологический стек (Sprint 1)
* **Frontend**: React / TypeScript (типизация данных)
* **Backend**: Node.js / Python (проектирование API)
* **База данных**: PostgreSQL / MongoDB (проектирование схемы)
* **Документация**: Markdown, архитектурные схемы

## Как запустить (Frontend)
1. Перейти в папку: `cd portfolio-builder-fe`
2. Установить зависимости: `npm install`
3. Запустить проект: `npm run dev`
